Esto es un simulador interactivo de la fase de grupos de la Copa del Mundo Qatar 2022.

- En el mismo se podrá ver la cuenta regresiva para el comienzo del mundial.
- Se podrán sortear los resultados de cada partido calculando los puntos correspondientes.
- También están cargadas todas las alineaciones de cada equipo para visualizar y comparar en cada partido.

* Nota: el proyecto cuenta con 3 archivos .js; aunque también se puede encontrar 
código Javascript escrito con la etiqueta "<script></script>" dentro de los archivos .html.
Considero que de esta manera se facilita la comprension, debido a las funciones que contiene el
 código del proyecto.
